# Changelog

All notable changes to EdgeBoard will be documented in this file.

## [1.0.0] - 2026-01-20

### Added
- 🎨 实时多人协作白板功能
- 🌈 8 种预设颜色选择
- 📏 可调节画笔粗细（1-20px）
- 🖱️ 支持鼠标和触摸屏绘画
- 🔄 实时笔画同步机制
- 🗑️ 全局画布清空功能
- 📊 连接状态和统计面板
- 🐛 调试日志查看器
- ❓ 内置帮助面板
- 📱 响应式设计，完美支持移动端
- 🎯 基于阿里云 ESA 边缘函数的无服务器架构
- 💾 EdgeKV 存储支持
- 🔒 原生安全防护（DDoS + WAF）
- 🌍 全球 3200+ 节点低延迟访问
- 📦 开箱即用，无需配置
- 📝 完整的文档和部署指南
- 🎁 MIT 开源许可证

### Technical Highlights
- React 18 + Vite 5 现代化技术栈
- Canvas API 高性能绘图
- Tailwind CSS 现代化样式
- 轮询同步机制（发现 → 拉取 → 绘制）
- 防竞态设计（独立笔画存储）
- 错误处理和自动重试
- 批量数据传输优化

### Infrastructure
- 阿里云 ESA 边缘函数部署
- EdgeKV 分布式存储
- 全球 CDN 加速
- 自动 HTTPS
- 免费套餐支持
