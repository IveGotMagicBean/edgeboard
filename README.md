# 实时协作白板 🎨

基于 React + Vite + 阿里云 ESA 的极简实时协作白板应用

## ✨ 功能

- 🎨 实时多人协作绘图
- 🌈 多种颜色选择
- 📏 画笔粗细调节
- 🧹 一键清空画布
- ⚡ 100ms 低延迟同步

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 📁 项目结构

```
realtime-whiteboard/
├── index.html           ← HTML 入口（根目录）
├── src/
│   ├── App.jsx         ← 主应用
│   ├── main.jsx        ← React 入口
│   └── index.css       ← 样式
├── package.json
├── vite.config.js
└── tailwind.config.js
```

## 🌐 部署

支持部署到：
- Vercel
- Netlify
- 阿里云 ESA
- GitHub Pages
